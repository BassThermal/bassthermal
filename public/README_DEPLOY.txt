BassThermal static site package v4-seo

Copy the contents of /public into the repository's public/static root.
Do not overwrite an existing real app-ads.txt unless you intentionally want to replace it.
This package includes homepage, /apps, app pages, /privacy, /support, robots.txt, sitemap.xml, stylesheet, and OG PNG assets.

After deploy:
1. Verify https://bassthermal.com/
2. Verify https://bassthermal.com/apps/
3. Verify https://bassthermal.com/sitemap.xml
4. Submit sitemap in Google Search Console.
