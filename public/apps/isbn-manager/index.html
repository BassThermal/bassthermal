BASSTHERMAL / apps / isbn manager
ISBN Manager book inventory and barcode label app
summary
ISBN Manager helps turn messy book lists into cleaner inventory records. Look up ISBN metadata, review and edit book fields, generate barcode labels, and export organized records for small selling, cataloging, or collection workflows.
workflow
  1. Enter, paste, import, or scan ISBNs.
  2. Review fetched metadata and clean inventory fields.
  3. Generate barcode labels and export organized records.
use cases
  • clean up used book inventory
  • look up book metadata from ISBNs
  • prepare barcode labels for book records
  • export organized catalog data
who for
For small booksellers, collectors, church libraries, school projects, and anyone cleaning up book inventory from ISBNs.
