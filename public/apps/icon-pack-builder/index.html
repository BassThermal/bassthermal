Icon Pack Builder

Generate Windows app, installer, and Microsoft Store icon assets from one source image.

windows
overview
Icon Pack Builder turns one source PNG into the Windows app icon sizes and package assets needed for app builds, Microsoft Store submissions, and installers. It keeps the output organized so you can avoid repeated manual resizing before each release.
workflow
  1. Load a master icon image.
  2. Generate the required icon sizes and package assets.
  3. Export the icon pack for app packaging or store submission.
use cases
  • generate Windows app icon sizes from one image
  • prepare Microsoft Store icon assets
  • create installer and desktop app icons
  • avoid manually resizing the same logo repeatedly
who for
For indie developers and app builders who need clean icon outputs from one source image instead of manually resizing the same logo over and over.
