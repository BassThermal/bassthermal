BASSTHERMAL / Website Image Inventory
Website Image Inventory website image audit tool
summary
Website Image Inventory scans public websites and creates a clean, reviewable inventory of discovered image assets for auditing, research, migration, and web-development workflows.
workflow
  1. Enter a public website or starting URL.
  2. Scan the site and collect discovered image references and metadata.
  3. Review, organize, and export the resulting image inventory.
use cases
  • audit images used across a public website
  • inventory image URLs and metadata
  • prepare for website migration or redesign
  • review discovered web assets without recording them manually
who for
For web developers, designers, site owners, researchers, and anyone who needs a structured view of images discovered across a public website.
